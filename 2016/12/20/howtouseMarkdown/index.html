Markdown语法

发表于 2016-12-20 |

1、标题,注意#与文字之间加一个空格

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

展示效果：

#一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

2、列表

###无序列表（用* 或者 -）

* 简介
* 第一章
* 结束语

展示效果：

  • 简介
  • 第一章
  • 结束语

###有序列表（用1. 2. 3.）

1. 第一条
2. 第二条
3. 第三条

展示效果：

  1. 第一条
  2. 第二条
  3. 第三条

3、引用

>  这里是引用啊

展示效果：

这里是引用啊

4、图片与链接

格式:[描述](链接)  图片在前面加!

[Mac上MarkDown工具推荐 ：Mou](http://25.io/mou/)

![M](https://avatars2.githubusercontent.com/u/2938786?v=3&u=bfbfd2920ad7fcf69e16db28c68d58ad84e39cfd&s=140)

展示效果：
Mac上MarkDown工具推荐 ：Mou

5、粗体与斜体（*包围是粗体 包围是斜体）

**这里是粗体** *这里是斜体*

展示效果：
这里是粗体 这里是斜体

6、代码

单行代码使用`包围
`git push origin master`

多行代码可以 在开始处空四个空格即可

展示效果：

git push origin master

while(true){
    print("MarkDown is cool")
}

7、表格

|   Column1Name   |   Column2Name   |   Column3Name   |
| --------------- |:---------------:| ---------------:|
|      leftAlign  | centerAlign     |   rightAlign    |
|      leftAlign  | centerAlign     |   rightAlign    |
|      leftAlign  |  centerAlign    |   rightAlign    |

展示效果

Column1Name Column2Name Column3Name
leftAlign centerAlign rightAlign
leftAlign centerAlign rightAlign
leftAlign centerAlign rightAlign

8、字体大小颜色

<font color=#0099ff size=5 face="黑体">color=#0099ff size=5 face="黑体"</font>

size：规定文本的尺寸大小。可能的值：从 1 到 7 的数字。浏览器默认值是 3。

展示效果：

color=#0099ff size=5 face=”黑体”

9、其他

单个回车视为空格。
连续回车才能分段。
行尾加两个空格，即可段内换行。
*** 表示分割线
符号要和文字之间加上一个字符的空格
